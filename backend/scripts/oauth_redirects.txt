--------------------------------------------------------------------------------
GOOGLE OAUTH REDIRECT (Gmail Integration)
--------------------------------------------------------------------------------

Redirect URI:
https://intelligent-email-assistant-3e1a.onrender.com/auth/google/callback

Configuration Steps:
1. Go to: https://console.cloud.google.com/apis/credentials
2. Select your OAuth 2.0 Client ID
3. Under "Authorized redirect URIs", add:
   https://intelligent-email-assistant-3e1a.onrender.com/auth/google/callback
4. Save changes
5. Verify GOOGLE_CLIENT_ID and GOOGLE_CLIENT_SECRET in Render environment variables

--------------------------------------------------------------------------------
MICROSOFT OAUTH REDIRECT (Outlook Integration)
--------------------------------------------------------------------------------

Redirect URI:
https://intelligent-email-assistant-3e1a.onrender.com/auth/microsoft/callback

Configuration Steps:
1. Go to: https://portal.azure.com/#blade/Microsoft_AAD_RegisteredApps/ApplicationsListBlade
2. Select your App Registration
3. Navigate to "Authentication" → "Platform configurations" → "Web"
4. Under "Redirect URIs", add:
   https://intelligent-email-assistant-3e1a.onrender.com/auth/microsoft/callback
5. Save changes
6. Verify OUTLOOK_CLIENT_ID and OUTLOOK_CLIENT_SECRET in Render environment variables
